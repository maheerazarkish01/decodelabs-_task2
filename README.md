# 🚀 Asteroid Risk Prediction & Dashboard

## 📌 Project Overview
This project predicts whether an asteroid is Dangerous or Safe using Machine Learning.

## 🎯 Features
- Data analysis
- ML model (Decision Tree)
- Streamlit dashboard
- Real-time prediction

## ▶️ How to Run
pip install -r requirements.txt
streamlit run app.py

## 🛠️ Technologies
Python, Pandas, Matplotlib, Scikit-learn, Streamlit
